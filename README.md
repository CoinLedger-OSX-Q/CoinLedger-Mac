## Overview

<a href="#">CoinLedger</a> handles the categories of cryptocurrency activity that create the most complexity for tax purposes — DeFi interactions, NFT trading, liquidity pool participation, and cross-chain activity. These transaction types require interpretation before tax treatment applies, and CoinLedger handles that interpretation automatically.

<a href="#">DeFi protocol support</a> categorizes liquidity provision, yield farming, lending, borrowing, and token swaps through decentralized exchanges. Each interaction type receives the appropriate tax classification rather than being lumped together as generic trades. <a href="#">NFT transaction tracking</a> records purchase prices, sale proceeds, and royalty income from non-fungible token activity with capital gains calculation for each disposal.

The <a href="#">cost basis methods</a> available include FIFO, LIFO, HIFO, and specific identification, with jurisdiction-aware defaults that apply the method required or recommended for your tax location. <a href="#">Tax loss harvesting analysis</a> identifies positions with unrealized losses that could be sold to offset gains elsewhere in the portfolio before year-end closes.

<a href="#">Report generation</a> produces jurisdiction-specific output formats including the forms used in the US, UK, Canada, Australia, and other major markets. Generated reports integrate with popular tax filing software. <a href="#">Audit documentation</a> maintains complete transaction-level records linking every figure in the tax report back to specific blockchain transactions and exchange records. The combination of comprehensive import, accurate categorization, and professional report output makes CoinLedger the complete solution for <a href="#">cryptocurrency tax compliance</a> on Mac.
